# Ant Design Pro

This project can help developers to create new project

## Environment Prepare

creatcli -v `查看版本号`:
creatcli -h `查看帮助文档`:
creatcli init `开始创建`:


```bash
creatcli init
```

then

```bash
cd your projectName
```


```bash
npm install
```
or

```bash
yarn
```



## Provided Scripts

Creatcli provides some useful script to help you quick start and build with web project and data screen project and mobile project.


### Start project

```bash
npm start
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
